## 🚗 Car Rental Platform

A full-stack MERN car rental platform supporting 50+ vehicles with real-time booking, location-based discovery, and token-based authentication.
Features an admin dashboard to manage 100+ bookings, with ImageKit CDN integration reducing asset load times by 40%.

**Tech Stack:** MongoDB · Express.js · React.js · Node.js · JWT · ImageKit

**live preview:** https://car-rental-gs.vercel.app
