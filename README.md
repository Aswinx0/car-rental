# AI powered Car Rental Fullstack

A full-stack car rental application with real-time booking, location-based discovery, and an AI-powered chatbot for car recommendations.

## ✨ Features
- 🔍 Location-based car discovery
- 📅 Real-time booking with double-booking prevention
- 🔐 JWT-based authentication
- 🖼️ ImageKit CDN integration (~40% faster asset loading)
- 🤖 AI chatbot powered by RAG + LLaMA 3 for smart car recommendations
- 📊 Admin dashboard for managing bookings and fleet

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, Context API
**Backend:** Node.js, Express.js, MongoDB, Mongoose
**Auth:** JWT (JSON Web Tokens)
**AI/ML:** LangChain, Groq (LLaMA 3), FAISS, Cohere Embeddings
**Media:** ImageKit CDN


## Project structure

- `client/` - React app built with Vite
- `server/` - Express backend API
